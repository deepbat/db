# Deepak Ultimate Portfolio

A clean, professional personal portfolio styled after the University of Bath website — light theme, strong typography, Bath purple (`#512886`) accent, and card-based content blocks.

## Features
- Bath-style **utility bar** + sticky navigation with mobile hamburger menu
- **Hero** with muted autoplay video background and purple gradient overlay
- **Highlights** stat strip (like Bath's "Result highlights")
- About, Experience cards, Skills with progress bars, Projects with images
- **Showreel** video section (`images/20260301_011203-CINEMATIC.mp4`)
- **Gallery** with lightbox (all photos in `images/`)
- **Achievements** highlight cards
- Structured multi-column footer
- Scroll-reveal animations, back-to-top, fully responsive, respects `prefers-reduced-motion`

## Files
- `index.html` — structure & content
- `style.css` — Bath purple light theme & layout
- `script.js` — menu, scroll reveal, video, gallery lightbox
- `images/` — photos & cinematic video

## Usage
Open the folder in Visual Studio Code and use the **Live Server** extension, or just open `index.html`.

## Customize
1. Replace `your@email.com` and phone in the utility bar, contact section and footer.
2. Edit placeholder text in About / Experience / Projects / Achievements.
3. Swap the hero video/poster or use a still image if you prefer no autoplay.
4. Add or remove photos by editing the `.g-item` links in the Gallery section.
5. Publish on GitHub Pages.
