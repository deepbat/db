# Deepak Ultimate Portfolio

A modern, responsive personal portfolio website.

## Features
- Animated gradient background + soft cursor glow
- Sticky glassmorphism navigation with mobile hamburger menu
- Scroll-reveal animations (IntersectionObserver)
- About, Experience timeline, Skills with progress bars, Projects
- **Showreel** video section (`images/20260301_011203-CINEMATIC.mp4`)
- **Gallery** with lightbox (all photos in `images/`)
- Back-to-top button & smooth scrolling
- Fully responsive & respects `prefers-reduced-motion`

## Files
- `index.html` — structure & content
- `style.css` — styling & animations
- `script.js` — menu, scroll reveal, video, gallery lightbox
- `images/` — photos & cinematic video

## Usage
Open the folder in Visual Studio Code and use the **Live Server** extension, or just open `index.html`.

## Customize
1. Replace `your@email.com` and phone in `#contact`.
2. Edit placeholder text in About / Experience / Projects.
3. Add or remove photos by editing the `.g-item` links in the Gallery section.
4. Publish on GitHub Pages.
