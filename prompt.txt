Create an interactive underwater ocean floor simulation website with the following features:

1. **Long-press sculpting**: User long-presses and drags to sculpt the ocean floor terrain - digging pools (lowering terrain) and raising ridges (elevating terrain).

2. **Tap ripples**: User taps anywhere to drop water ripples that propagate across the terrain.

3. **Liquid lens effect**: Ripples bend, speed up, and focus around the underwater terrain features, behaving like liquid lenses that interact with the sculpted topography.

4. **Visual style**: Beautiful underwater aesthetic with realistic water effects, depth-based coloring, and smooth animations.

5. **Tech stack**: HTML5 Canvas or WebGL for rendering, vanilla JavaScript for physics simulation.
